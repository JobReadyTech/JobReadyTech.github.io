MS SQL Server 2019. Programación de Bases de Datos con T-SQL

Con este curso de Microsoft SQL Server 2019, partiremos de cero, para mostrar este gestor de bases de datos, al mismo tiempo que nos formaremos con SQL y la extensión de funcionalidades del lenguaje que hace Microsoft, llamado Transact-SQL.

Veremos todos los comandos y sentencias necesarios para poder trabajar con fluidez en un proyecto.

Desarrollaremos proyectos de bases de datos para poder trabajar elaboración de informes completos o scripts que puedan ser integrados en otros framework's de desarrollo.

El curso no se cierra, sino que quedará abierto a secciones que expliquen temas más específicos, dentro de este nivel.

- Comenzaremos con unas pautas de instalación de SQL Server y Management Studio.

- A continuación, unas clases teóricas opcionales pero recomendables sobre nociones de diseño de Bases de Datos.

- A partir de aquí, una relación de clases con los comandos más utilizados, comenzando con los que seleccionan, insertan, modifican y borrar datos de una base de datos.

- Todas las clases se basarán en proyectos completos de gestión de una base de datos

Dispondréis de ejercicios en la mayoría de secciones, así como unos exámenes de test para reforzar lo aprendido.

Enero 2021: Ampliación de dos secciones más con 27 clases. Duración total del curso. 16 horas

En esta ampliación haremos hincapié en el trabajo con procedimientos almacenados y funciones para la gestión de un base de datos con múltiples tablas

El curso se revisará para la modificación de clases que deban actualizarse; asimismo se incluirán periódicamente clases de tipo anexo, con funcionalidades que puedan ser de interés.
